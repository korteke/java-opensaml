Changes in Release 1.2.0
=============================================
* Add StatusMessage and Description XACML object implementations
* Fix JOST-38
* Correct how XML boolean values are dealt with in XACML objects
* Deprecate FileBackedHTTP, Filesystem, and HTTP metadata providers in favor of new ResourceBacked metadata provider.
  This provides a uniform layer for going out and fetching remote resources.
